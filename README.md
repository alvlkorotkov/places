# Places Landing

Базовая структура проекта под лендинг по задаче генерации пространства для проекта "Вкус крови".

## Структура

- `index.html` — точка входа лендинга
- `src/styles/main.css` — основные стили
- `src/scripts/main.js` — клиентская логика и интерактив
- `src/content/site-copy.md` — черновик текстов и структуры блоков
- `docs/brief.md` — рабочий бриф и договоренности по задаче
- `assets/images` — изображения для лендинга
- `assets/references` — референсы, мудборды, исходники

## Дальше

1. Зафиксировать бриф и цель страницы в `docs/brief.md`.
2. Собрать тексты блоков в `src/content/site-copy.md`.
3. После этого перейти к дизайну и верстке в `index.html` и `src/styles/main.css`.
